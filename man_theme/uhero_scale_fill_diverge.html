uhero_scale_fill_divergeR Documentation

Applies the UHERO color palette to ggplot's diverging fill aesthetic.

Description

Uses ggplot2::scale_fill_gradient2

Usage

uhero_scale_fill_diverge(high = "blue", low = "orange", ...)

Arguments

high

A string that defaults to "blue". Can accept any of the other colors in uhero_color_list.

low

A string that defaults to "orange". Can accept any of the other colors in uhero_color_list.

...

Any other parameters that can be passed to ggplot2::scale_fill_gradient2
