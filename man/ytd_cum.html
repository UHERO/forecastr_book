ytd_cumR Documentation

Year to date sum or average

Description

Year to date sum or average

Usage

ytd_cum(x, avg = TRUE)

Arguments

x

a ts-boxable object

avg

if TRUE (default), return year to date average, if FALSE, return year to date sum

Value

object of the same type as the input containing year to date sum or average

Examples

monthly_data_example |>
  ytd_cum()
monthly_data_example |>
  tsbox::ts_long() |>
  tsbox::ts_pick("VISNS_HI") |>
  tsbox::ts_xts() |>
  ytd_cum(avg = FALSE) |>
  tsbox::ts_plot()
