Convert dates from yyyy-mm-dd to yyyyQqq format

Description

Convert dates from yyyy-mm-dd to yyyyQqq format

Usage

ymd_to_yQq(x)

Arguments

x

dates (string: yyyy-mm-dd)

Value

formatted dates (string: yyyyQqq)

Examples

ymd_to_yQq(c("2010-01-01", "2020-10-01"))
ymd_to_yQq(c("2010-01-01", "2020-10-01")) |> lubridate::yq()
